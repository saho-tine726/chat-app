## 解説・工夫した点など
* デザインも自作
* Firebaseにデータを格納
* Firebase FirestoreのonSnapshotを使い、データの追加をリアルタイムで反映
* Googleアカウントのログイン機能を搭載
* React hooks（カスタムフック）使用
* CSSはSCSS、一部Marerial Ulで記述
